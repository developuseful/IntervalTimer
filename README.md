# ⏱️ Interval Timer (Test Task)

Android-приложение для интервальных тренировок с точным таймером и поддержкой работы в фоне.

---

## 🚀 Функциональность

- Загрузка тренировки по ID (API)
- Отображение списка интервалов
- Таймер:
  - ▶️ Старт  
  - ⏸️ Пауза  
  - ⏯️ Продолжение  
  - 🔄 Сброс
- 🔊 Звуковые сигналы:
  - старт
  - смена интервала
  - завершение тренировки
- 📳 Работа в фоне через **Foreground Service**
- 🔔 Постоянное уведомление с текущим состоянием таймера
- ♻️ Восстановление состояния после возврата в приложение

---

## 🛠️ Стек

- **Kotlin**
- **Jetpack Compose**
- **MVVM**
- **Coroutines + Flow**
- **Retrofit**
- **Foreground Service**

---

## 🧠 Особенности реализации

- Бизнес-логика вынесена в `domain` слой
- UI подписывается на `StateFlow` из ViewModel
- Используется **Foreground Service** для стабильной работы таймера в фоне
- Синхронизация состояния между Service и UI
